```bash

```

## Install
```bash
apt update && apt upgrade
apt install nodejs git ffmpeg
git clone https://github.com/NezukoChans28/nodebot
npm install
node index.js
Tinggal Scan QR Code Pake Hp Yang Bua Jalanin Bot
```
